Нейроночки

Ассистенты:

  • DeepSeek (chat.deepseek.com): Просто китайский папочка, все умные все знают, юзать лучше R1
  • Perplexity (perplexity.ai): поисковая система с ИИ и помощник в исследованиях
  • Claude (claude.ai): универсальный чат-бот, подходит для проектов и обмена работами
  • ChatGPT (chatgpt.com): кому я рассказываю...., но волки советуют Advanced Voice Mode для общения с ИИ

Для продуктивной работы:

  • Granola (granola.ai): ИИ-блокнот, который слушает ваши встречи
  • Flow (flowvoice.ai): голосовая диктовка в любом приложении
  • Gamma (gamma.app) и SlidesAI (slidesai.io): создание презентаций
  • Adobe Acrobat (acrobat.adobe.com): работа с PDF-файлами
  • Cubby (cubby.nyc): пространство для совместных исследований
  • Cora (cora.computer): помощник для электронной почты
  • Lindy (lindy.ai): создание ИИ-агентов

Для создания продуктов (код):

  • Cursor (cursor.so): редактор кода с ИИ
  • Replit (replit.com): создание приложений из описания
  • Anychat (huggingface.co): использование любых ИИ-моделей
  • Codeium (codeium.com): автозаполнение кода

Для создания контента:

  • ElevenLabs (elevenlabs.io): реалистичные ИИ-голоса
  • Suno (suno.com) и Udio (udio.com): создание музыки
  • Midjourney (midjourney.com), Ideogram (ideogram.ai), Playground (playground.com): генерация изображений
  • Runway (runwayml.com), Kling (klingai.com), Viggle (viggle.ai): генерация видео
  • Krea (krea.ai): редактирование изображений и видео
  • PhotoRoom (photoroom.com): редактор для товаров

Эм, чем ты болен чтоб это юзать?
